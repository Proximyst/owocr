class OwO::Exceptions::Unauthorized

  • OwO::Exceptions::Unauthorized
  • Exception
  • Reference
  • Object

Overview

Unauthorized is raised when the token is deactivated at the OwO API. This can also mean the token was miswritten.

NOTE This is raised if a token does not have the required dashes or has whitespace.
